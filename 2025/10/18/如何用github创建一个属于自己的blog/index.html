如何用github创建一个属于自己的blog

无标签
发布日期:   2025-10-18

1.环境准备
安装git:
官网安装后在所在文件夹下右击选择Git Bash Here打开git
nodejs:

hexo:
git中输入命令行(在桌面右击或者想要存放博客的文件夹下打开都行)

npm install hexo -g

注意:git中粘贴复制选中后右击

2.git配置SSH key（免密上传本地资源到github中）
注册github账号（选个好点的名字）
查看本身是否配置SSH key（一般都没配置）

cd ~/.ssh

输入生成命令

ssh-keygen -t rsa -C “邮件地址”(注册github的邮箱)

直接回车3次
进入文件夹下的id_rsa.pub,用记事本打开，复制内容
点开github个人主页中的setting->SSH and GPG keys->New SSH key
Title名字随便，把复制的内容复制到Key中，点击Add SSH key
测试是否成功

ssh -T git@github.com

输入yes，当出现如下文字说明已经成功连接了

3 搭建个人博客

常用命令

hexo n “文章名字” 或者 hexo new “文章名字” #新建文章
hexo new page “页面名字” #新建页面
hexo g 或者 hexo generate #生成静态页面至public目录
hexo s 或者 hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo d 或者 hexo deploy #部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本

在想要存放博客的文件夹下打开git,初始化加相关配置

hexo init

git config –global user.name “用户名” #你的github用户名

git config –global user.email “邮箱” #填写你的github注册邮箱

生成静态网页+本地预览

hexo g
hexo s

或者hexo g && hexo s

在浏览器访问：

http://localhost:4000/

4 部署到个人GitHub（成功部署后小伙伴也可以通过用户名.github.io访问到你搭建的博客）
新建一个github仓库,点击New repository

仓库名字格式: 用户名.github.io(注意如果想一次运行成功就先这么起名,后面可以尝试New organization，或者修改_config.yml中root或者url,但我尝试第二种方法没成功)

编辑_config.yml

deploy:
type: git
repository: git@github.com:用户名/用户名.github.io.git
branch: main

url: https//用户名.github.io
root: /

发布到github

hexo g -d（如果想更快就hexo clean && hexo g -d）

出现报错HEAD -＞ main (push declined due to repository rule violations)

点击 To push, remove secret from commit(s) or follow this URL to allow the secret 下给出的链接，点击同意推送即可。

在浏览器访问

用户名.github.io

恭喜成功搭建，可以让小伙伴来访问

5 编写博客

hexo new ‘文章名’（文件在source->_posts->文章名.md）
等待一会，可以ctrl+F5强制刷新

下载一款编辑md的软件,这里我使用的是VScode,插件如下

Markdown Preview Enhanced,Markdown All in One和markdownlint

6 更换主题

github下载你喜欢的主题，解压到themes文件夹下，更改文件夹为主题名
将_config.landscape.yml修改为_config.主题名.yml，并将GitHub上的yml内容替换_config.主题名.yml
_config.yml中themes后面改为主题名
hexo d
