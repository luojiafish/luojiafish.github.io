大学生如何去找实习

Yuan hao2025-10-202025-12-05

目标：大学生阶段找到一份大厂实习（以电机控制实习岗为例）

希望对求职的经历进行总结

前期准备

  1. 简历投递（先以中小厂为主，在没有理论实践经验不要开始投，先去了解要求，去学习相关知识和实践项目）

  2. 个人项目demo准备

我目前是打算按照研究生研究方向来准备，完成研究生提前学习的同时，提前去企业了解方向未来发展

🧩 第一阶段：电机控制核心理论（建议 7 天左右）

总体目标（这一阶段结束你要能做到）：

  1. 口头讲清楚什么是 FOC（磁场定向控制），为什么要用 dq 坐标系。
  2. 画出完整的 FOC 框图（三环结构、电流/速度/位置环）。
  3. 能看懂常见的 FOC 资料和代码中的公式。

下面给你拆成 每 1–2 天一个块，每块给出：
-【要学会的知识点】
-【推荐书/文档/视频】（中英都有，优先 B 站 + 应用手册）

📆 Day 1–2：PMSM/BLDC 电机原理 + 数学模型

要学会的知识目标：

  • 理解三相 PMSM / BLDC 的结构、定转子磁场概念。
  • 知道“电角度、极对数、反电动势、转矩与电流的关系”。
  • 了解最基本的 abc 三相 → αβ → dq 变换前的电机方程长什么样（不用推得很细，但要知道电流/电压/转矩之间的大致关系）。

推荐资料：

  1. 视频（中文）——入门类

    • B 站《手把手教你驱动永磁同步电机》系列，前几讲是电机理论+FOC思想，非常适合你这种已有基础的人作为系统梳理。(哔哩哔哩)
    • B 站《【野火】电机系列教学视频，基于 STM32 硬件》基础部分（步进/直流/无刷/PMSM 概览），尤其是总览和电机原理部分。(哔哩哔哩)

  2. 文档（英文，偏公式）

    • TI 应用报告 “Sensored FOC of PMSM”（SPRA 之类的文档）：里面有标准 PMSM 数学模型和 FOC 框图，你只需要先看前两章：电机原理 + 基本方程。(e2e.ti.com)

✅ 这两天你可以一边看 B 站视频，一边对照 TI PDF 里的框图，把“电流、电压、磁链、转矩”之间的关系搞清楚。

✅ 建议做一页纸总结：画出 PMSM 的简化等效电路 + 转矩公式。

📆 Day 3–4：Clarke / Park 变换 + dq 坐标系 + FOC 思想

要学会的知识目标：

  • 明确：

    • Clarke 变换：abc → αβ（静止坐标系）
    • Park 变换：αβ → dq（转子同步旋转坐标系）

  • 理解 dq 坐标的直观意义：“把三相交流电流投影到转子磁场上，变成类似直流的两个分量 Id / Iq”。

  • 能解释：

    • “为什么在 dq 坐标下，可以做到转矩/磁链解耦，像直流电机一样控制？”

推荐资料：

  1. 视频：

    • 继续看《手把手教你驱动永磁同步电机》中专门讲 Clarke / Park 变换、dq 电流的几讲。(哔哩哔哩)
    • B 站上搜“电流、速度、位置三闭环 / 三环控制、FOC”，很多短视频会用图示解释三环结构。(哔哩哔哩)

  2. 文档 / 书：

    • NXP 的应用手册 “Field-Oriented Control (FOC) of 3-Phase PMSM and BLDC Motors”，里面有比较标准的 dq 变换推导和 FOC 框图，可以跳读数学细节，重点看概念图。(NXP半导体)
    • Microchip / Microchip 相关 FOC 应用手册中对“把 AC 变 DC”的那部分说明。(Microchip)

✅ 这一块结束后，你要能在纸上写出 Park 变换矩阵，并且用口语解释：“dq 坐标系就是跟着转子一起转的参考系，因此在这个坐标里，看见的是直流分量”。

📆 Day 5：SVPWM（空间矢量 PWM）

要学会的知识目标：

  • 了解传统 SPWM 的思想（正弦波调制）。

  • 理解 SVPWM 的核心：

    • 把三相逆变器看成 8 个电压矢量
    • 根据目标电压矢量所在扇区，分配 T1、T2、T0 占空比

  • 知道 SVPWM 相比 SPWM 的优势：直流母线利用率高、谐波更好等。

  • 不要求你把数学推导得很细，但要理解扇区划分、占空比组合的概念。

推荐资料：

  • B 站上搜 “SVPWM 原理 FOC 电机控制”，挑 1–2 个画图讲得清楚的看完一遍（只要有扇区图+波形图就够）。
  • 很多 FOC 应用手册（TI / NXP）里都有 SVPWM 一章，可以翻一下示意图。(e2e.ti.com)

✅ Day 5 的输出：画一张“六扇区 + 电压矢量 + T1/T2/T0”的图，自己能对着图讲出来。

📆 Day 6：三环控制结构（电流环、速度环、位置环）+ PID 思路

要学会的知识目标：

  • 理解三环的作用和时间尺度：

    • 电流环：最快，保证电流跟踪
    • 速度环：中速，调速
    • 位置环：最慢，做定位 / 轨迹

  • 三环如何串联：位置给定 → 速度给定 → 电流参考 → dq 电流 PI → SVPWM。

  • 大致知道 PID / PI 在这里的角色（补偿误差、稳定系统），会调“比例 P、积分 I”的基本方向。

推荐资料：

  • B 站“闭环步进电机三环控制（电流环，速度环，位置环）演示”、“三闭环 FOC”之类短视频，看思路和波形即可。(哔哩哔哩)
  • 你以前看过阶跃响应 PID，那些知识可以直接迁移到“速度闭环 / 电流闭环”。

✅ 这一块结束，你要能画出 完整 FOC 三环控制框图，并且能对每一环做一句话说明。

📆 Day 7：系统复盘 + 选定后续主线参考课程

这一天不看新东西，只做整理：

  1. 把这一周的内容整理成 2–3 页的 PDF 或笔记（后面可以直接作为面试讲稿）。
  2. 选定 后面要跟的主要视频/文档主线，建议：

  • Simulink / 仿真主线：

    • MathWorks 的 “Field-Oriented Control of PMSMs with Simulink” 视频系列（估计你是学生，学校有 Matlab 授权的话可以直接看）(MathWorks)
    • 配合 GitHub 上的 Simulink FOC 仿真项目（后面第二阶段会用到）。(GitHub)

  • STM32 / 实战主线：

    • B 站《STM32G4 FOC 开发实战》系列，它从底层驱动到 Simulink 算法模型都有，非常贴近你要做的事。(哔哩哔哩)
    • 或者《手把手教你驱动永磁同步电机》继续跟到底。(哔哩哔哩)

🧩 第二阶段：Simulink 仿真项目（先告诉你项目名 + 对应 GitHub）

你现在先不用看细节实现，先把“项目选型”定下来，后面你问我再一起拆模块。

✅ 项目 2-1：《PMSM FOC 速度控制 Simulink 实现》

  • 内容：
    用 Simulink 搭建一个 PMSM 的 FOC 控制系统，包含：PMSM 模型、Clarke/Park、dq 电流 PI、速度 PI、SVPWM、速度给定与负载扰动等。

  • 开源参考：

    • GitHub FOC_PMSM：一个完整的 PMSM FOC Simulink 模型，含参数脚本、模型文件和说明。(GitHub)
    • GitHub FOC_PMSM_Simulaion：结构清晰、注释完善的 FOC 仿真模型，可以直接跑。(GitHub)
    • GitHub PMSM-Controller-Simulink：基于 Simulink 实现的 PMSM FOC 控制器，带有文档。(GitHub)

✅ 项目 2-2：《PMSM 三环控制（电流/速度/位置）仿真》

  • 内容：
    在 2-1 的基础上增加位置环，实现“位置给定 → 速度给定 → 电流参考”的完整三闭环仿真，并观察不同环节参数对响应的影响。

  • 开源参考：

    • 上面那几个 FOC Simulink 仓库，有的 already 带有速度/位置控制案例，你可以在此基础上扩展。(GitHub)
    • Matlab 官方 demo / HIL 示例中也有包含位置控制的 FOC 模型，可以作为结构参考。(MathWorks)

🧩 第三阶段：STM32 FOC 工程项目（同样先给项目名 + GitHub）

✅ 项目 3-1：《基于 STM32F1 的简易 FOC 电机控制库（带 SVPWM）》

  • 内容：
    用 STM32F103/类似芯片实现一个简化版 FOC 控制框架：

    • 三相电流采样（哪怕先用虚拟值）
    • Clarke / Park 变换
    • dq 电流 PI
    • SVPWM 占空比 → TIM1 产生 6 路 PWM

  • 开源参考：

    • GitHub EBiCS_motor_FOC：在 STM32F103 上实现的 FOC 固件，代码比较清爽，适合作为学习模板。(GitHub)
    • GitHub DengFOC_on_STM32：跟着邓 FOC 教程在 STM32 上实现的例程。(GitHub)
    • GitHub STM32 SERVO FOC PMSM with Encoder：基于 STM32F103 + 编码器的伺服 FOC 工程。(GitHub)

✅ 项目 3-2：《基于 STM32 的通用 FOC / BLDC 控制框架移植与简化》

  • 内容：
    选一个成熟、稍大一点的 FOC 开源工程，挑其核心模块（控制算法部分）看懂、精简、做笔记，形成你自己的“二次封装 / 精简版控制框架”。

  • 开源参考：

    • GitHub MESC_Firmware：一个功能很全的基于 STM32 的 BLDC FOC 库，支持多种工作模式，很适合“拆解学习”。(GitHub)
    • GitHub STM32_SimpleFOC 或 stm32-FOC：基于 SimpleFOC 思想的 STM32 FOC 实现，用于步进/BLDC，结构清晰。(GitHub)

🧩 第四阶段：文档 / 简历 / 面试准备

你说要穿插在二三阶段中，这样安排比较好：

  • 在做 2-1 / 2-2 时：
    就开始截图、整理仿真曲线，写模型结构说明，将来直接塞进简历。
  • 在做 3-1 / 3-2 时：
    顺手整理好：工程目录结构、主要 C 文件说明、关键函数流程图。
