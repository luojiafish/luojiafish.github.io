基础知识

有刷电机和无刷电机的区别

有刷电机是把电枢线圈绕在转子上，通过电刷 + 换向器把电流送进转子线圈并完成换向，让转子通电带磁，定子一般是永磁体或励磁绕组提供固定磁场；

无刷电机则是把电枢线圈固定在定子上，转子做成永磁体，由电子驱动电路按一定顺序给定子绕组通电，在定子上产生“旋转磁场”，去拖动带永磁体的转子转动，因此不需要电刷。

图1(b) 有刷电机（转子电枢+电刷换向器）

补充说明：

  • 有刷电机的核心局限：电刷与换向器摩擦导致磨损、发热、噪音，寿命短（一般几千小时），不适合高速/长时运行；
  • 无刷电机的核心优势：无机械换向磨损，寿命长（数万小时）、效率高、噪音低，可通过电子控制实现高精度调速/力矩控制，是FOC控制的核心应用对象。

电子调速器（电调）

无刷电调（ESC）：你现在准备写的 FOC + SVPWM 程序，如果烧到一块 MCU + 三相桥的板子上，那整块板子对外的身份就是一个“电调”

补充说明：

  • 电调的核心功能：接收控制指令（如遥控器PWM、CAN信号）→ 处理电机反馈（电流/位置）→ 输出驱动信号（SVPWM）→ 控制电机运行；
  • 按控制算法分类：
    1. 传统电调：采用六步方波换相算法，仅支持转速开环/半闭环控制；
    2. FOC电调：采用磁场定向控制算法，支持电流（力矩）-速度-位置三闭环控制，性能更优（对应知乎文章核心内容）。

什么是FOC

FOC（Field-Oriented Control）：磁场定向控制，也被称作矢量控制（vector control）: 一种电机控制方法，是目前无刷直流电机（BLDC）和永磁同步电机（PMSM）高效控制的最优方法之一

FOC优势：采用 FOC 控制算法的电调，在低速控制、力矩控制、噪音等性能上，优于采用六步方波算法的传统电调

普通电调优势：

  1. 兼容性，同一电调不同的BLDC不用调参
  2. 便宜，成本低
  3. 算法简单，运算量少，很适合需要提高带宽的超高转速电机

补充说明（FOC优于六步方波的核心原因）：

  1. 转矩脉动小：六步方波通过“两两导通”换相，转矩随换相时刻突变（脉动率一般10%-20%）；FOC通过正弦电流驱动，转矩连续平滑（脉动率可降至5%以下），低速运行无抖动；
  2. 控制维度全：六步方波仅能间接控制转速，无法精准控力矩；FOC可独立控制d/q轴电流，实现转矩、转速、位置三闭环控制（对应知乎文章“三闭环”核心逻辑）；
  3. 效率更高：FOC能让定子磁场与转子磁场始终保持90°电角（最大转矩输出角度），相同电流下输出转矩更大，电能转化效率提升5%-15%；
  4. 低速性能好：六步方波低速时换相频率低，易出现“爬行”现象；FOC通过电流环快速响应，零速/低速可稳定输出恒定转矩（如机械臂抱持负载）。

图2 左手定则（磁场方向×电流方向=安培力方向，决定电机转矩方向）

右手定则：判断带点线切割磁场生成的反电动势方向


图3 右手定则（磁场方向×运动方向=感应电动势方向，反电动势阻碍电流，影响电机驱动电压）

补充说明：反电动势是FOC控制的关键物理量——电机转动时，定子绕组切割转子磁场产生反电动势（E=KeωE=K_e\omegaE=Ke​ω，KeK_eKe​为反电动势系数，ω\omegaω为电角速度），其幅值随转速升高而增大，因此高速驱动时需提高母线电压抵消反电动势。

右手螺旋定则：判断通电的线圈的极性


图4 右手螺旋定则（四指为电流方向，大拇指为线圈N极方向，决定定子磁场极性）

PWM波：本质是用面积等效来近似改变波的有效值

补充说明：

  • 电机驱动中PWM的核心作用：通过高频开关（一般10kHz-20kHz）控制MOS管导通时间，等效输出可变幅值的直流/交流电压；
  • 频率选择依据：过低会导致电机噪音大、转矩脉动明显；过高会增加MOS管开关损耗，降低电调效率，需结合电机电感和驱动芯片特性平衡。

无刷电机原理

根据磁铁同性相斥，异性相吸的原理，中间永磁体在两侧电磁铁的作用下会被施加一个力矩并发生旋转，这就是电机驱动的基本原理
对于简化的无刷电机来说，以三相二极内转子电机为例（如 图1(a)），每项引出一头，另一头与其他两项两两连接（定子的三相绕组有星形联结方式和三角联结方式，而三相星形联结的二二导通方式最为常用）,通过三项的两两导电，带动内部转子移动，有连续6个状态，过程像毛驴前面挂着个胡萝卜一样


图5 星型与三角型连接图

星形接法:

  1. 线电压是相电压的√3倍，线电流等于相电流。有助于降低绕组承受电压(220V)
  2. 星形接法缺点：电机功率减小，
  3. 小功率电机4KW以下的大部分采用星型接法

三角型接法:

  1. 有助于提高电机功率，
  2. 三角接法，缺点：启动电流大，绕组承受电压(380V)大增大了绝缘等级。

补充说明：

  • 无刷电机绕组连接选择：FOC控制中，星形接法更常用——相电流可通过母线采样电阻间接测量（成本低），且中性点电位稳定，利于电流闭环控制；三角接法虽功率密度高，但电流采样复杂，多用于大功率电机场景；
  • 二二导通的本质：六步方波的换相逻辑，每一步仅让两相绕组通电（如A+→B-），定子磁场跳变60°电角，转子跟随转动，这是传统电调的核心驱动方式。

直流电机 / 永磁直流电机 端电压 (U)、励磁磁通 (Φ\PhiΦ)、负载 电枢(电枢是进行能量转换、和外电路交换能量的绕组)电流 (IaI_aIa​) “励磁(励磁部分则负责产生主磁场)一定时：转速约 ∝∝∝ 电压，转矩 ∝∝∝ 电流”

BLDC 与 PMSM 的区别

因此为了电机转动抖动问题，从“硬件”和从“软件”出发有两个解决方案
BLDC:
方波驱动，靠在转子上多做几对磁极（多贴几块 N/S 磁铁），定子的绕组布置也会对应做成多极,来让电机低速更有力、位置划分更细一点。但BLDC由于反电动势接近梯形波，抖动是仍然存在的，不过不明显了

不懂，通过看书去补充知识

PMSM:
将方波驱动改为正弦波驱动，电机本身的磁场做得更像正弦，再配合“正弦电流 + 矢量控制 + SVPWM 这种更高级的 PWM”，
目的是让电机转起来更顺、更安静、力矩更平滑。

补充说明（BLDC与PMSM核心差异对照表）：

对比维度 BLDC（无刷直流电机） PMSM（永磁同步电机）
反电动势波形 梯形波（120°平顶） 正弦波
驱动方式 六步方波换相（两两导通） 正弦波驱动（FOC+SVPWM）
控制算法 基于霍尔传感器换相，逻辑简单 基于磁场定向控制，需Clark/Park变换
转矩脉动 中等（10%-20%） 小（<5%）
低速性能 一般（易爬行、抖动） 优秀（零速稳转矩）
核心应用场景 低成本设备（如无人机、电动工具） 高精度设备（如机械臂、伺服系统）

对三相 BLDC 来说，极对数从 1 增加到 p 后，
转子在一圈机械转动内会经历 p 个电角周期，
因此同样的霍尔/反电动势事件在机械角度上被细分了 p 倍。
控制器通过极对数 p 和霍尔/反电动势信号的时序与序列，来推断当前的电角位置，从而实现换相或 FOC 控制。

FOC整体框架

FOC控制原理

按照整体框架图（以电流闭环控制的BLDC）来看，FOC 控制的整个流程如下：

  1. 对电机三相电流进行采样，得到 Ia, Ib, Ic I_a,\ I_b,\ I_c Ia​, Ib​, Ic​

  2. 将 Ia, Ib, Ic I_a,\ I_b,\ I_c Ia​, Ib​, Ic​ 经过 Clarke 变换，得到 Iα, Iβ I_\alpha,\ I_\beta Iα​, Iβ​

  3. 将 Iα, Iβ I_\alpha,\ I_\beta Iα​, Iβ​ 经过 Park 变换，得到 Iq, Id I_q,\ I_d Iq​, Id​

  4. 计算 Iq, Id I_q,\ I_d Iq​, Id​ 与其设定值 Iq_ref, Id_ref I_{q\_ref},\ I_{d\_ref} Iq_ref​, Id_ref​ 之间的误差

  5. 将上述误差输入两个 PID（实际只使用 PI）控制器，得到输出的控制电压 Uq, Ud U_q,\ U_d Uq​, Ud​

  6. 将 Uq, Ud U_q,\ U_d Uq​, Ud​ 进行 逆 Park 变换，得到 Uα, Uβ U_\alpha,\ U_\beta Uα​, Uβ​

  7. 使用 Uα, Uβ U_\alpha,\ U_\beta Uα​, Uβ​ 合成电压空间矢量，输入 SVPWM 模块 进行调制，输出该时刻三个半桥的状态编码值（前文已提及）

  8. 根据前面输出的编码值，控制三相逆变器的 MOS 管开关，驱动电机转动

  9. 不断循环执行上述步骤，形成闭环控制

控制流程补充说明：

  • 步骤3的关键：Park变换需要转子实时电角度θ（有感由传感器提供，无感由算法估算），θ的精度直接决定d/q轴解耦效果——θ误差过大会导致转矩脉动增大、控制不稳定；
  • 步骤5的PI控制器：仅用PI不用D的原因——电流环响应速度快（带宽一般1kHz-5kHz），D项会放大采样噪声，导致电流抖动；PI的核心作用是“快速消除电流误差，输出所需的d/q轴电压”（对应电机d/q轴方程：Ud=RId+LddIddt−ωLqIqU_d=R I_d + L_d \frac{dI_d}{dt} - \omega L_q I_qUd​=RId​+Ld​dtdId​​−ωLq​Iq​，Uq=RIq+LqdIqdt+ωLdId+ωψfU_q=R I_q + L_q \frac{dI_q}{dt} + \omega L_d I_d + \omega \psi_fUq​=RIq​+Lq​dtdIq​​+ωLd​Id​+ωψf​）；
  • 步骤7的电压合成：Uα/Uβ是两相静止坐标系的电压指令，SVPWM通过选择相邻的两个基本电压矢量（如V1、V2）和零矢量（V0/V7），按伏秒平衡原理分配导通时间，等效合成目标电压矢量。

SVPWM技术深入理解

SPWM 的核心就是：用正弦参考波与高频载波（锯齿 / 三角）比较，把“正弦”编码成一串占空比随时间正弦变化的 PWM。

而SVPWM算法实际上计算的是图6所示逆变器的六个开关何时导通，何时切断。但是如何产生所需的正弦电压呢，PWM波等效而来的

所以你看到的 ±2π/3，既可以理解成 电角度上的相移，在物理上对应的是“绕组在空间上错开 120° + 电压在时间上错开 120°”。本质上就是用同一个电角度描述空间和时间的耦合。

补充说明（SVPWM核心细节）：

  1. 电压空间矢量的构成：三相逆变器的6个MOS管组合产生6个基本电压矢量（V1-V6，幅值为2Udc3\frac{2U_{dc}}{3}32Udc​​，相邻矢量夹角60°）和2个零矢量（V0：全下管导通；V7：全上管导通），所有矢量构成正六边形的“电压矢量圆”；
  2. 目标矢量的合成：任意目标电压矢量（如Uα/Uβ对应的矢量）都可由相邻的两个基本矢量和零矢量合成，通过计算三个矢量的导通时间（t1t1t1、t2t2t2、t0t0t0，满足t1+t2+t0=Tst1+t2+t0=T_st1+t2+t0=Ts​，TsT_sTs​为PWM周期），实现等效电压输出；
  3. SVPWM的优势：
    • 电压利用率高：相比SPWM，相同母线电压UdcU_{dc}Udc​下，SVPWM最大输出相电压为Udc3≈0.577Udc\frac{U_{dc}}{\sqrt{3}}≈0.577U_{dc}3​Udc​​≈0.577Udc​，SPWM仅为0.5Udc0.5U_{dc}0.5Udc​，提升15.47%；
    • 转矩更平滑：电压矢量连续旋转，无跳变，定子磁场平稳，电机运行噪音更低；
  4. 工程实现要点：需处理“过调制”问题——当目标电压矢量幅值超过基本矢量幅值时，需通过过调制算法修正，避免输出电压畸变。

关键疑问解答与补充

疑问1：能不能改进PID，让偏差能对应起来我想要的物理量，还是说没这个必要？

解答：
PI输出电压是FOC电流环的必然选择（非随意设计），但可通过优化逻辑让偏差更贴合物理量，无需改变输出维度（电压），具体如下：

  1. 为什么必须输出电压：逆变器能直接控制的物理量只有“定子电压”，电流的变化由电压决定（电机d/q轴方程），PI的作用是“根据电流误差，计算出消除误差所需的电压补偿量”，这是电机电磁特性决定的；
  2. 可优化方向（工程常用）：加入前馈补偿，让PI仅修正未知误差——
    原PI输出：Ud=KpdΔId+Kid∫ΔIddtU_d=K_{p_d}ΔI_d + K_{i_d}∫ΔI_d dtUd​=Kpd​​ΔId​+Kid​​∫ΔId​dt
    改进后：Ud=KpdΔId+Kid∫ΔIddt+ωLqIq+RId_refU_d=K_{p_d}ΔI_d + K_{i_d}∫ΔI_d dt + ω L_q I_q + R I_{d\_ref}Ud​=Kpd​​ΔId​+Kid​​∫ΔId​dt+ωLq​Iq​+RId_ref​
    前馈项（ωLqIq+RId_refω L_q I_q + R I_{d\_ref}ωLq​Iq​+RId_ref​）是电机方程中的已知压降（交叉耦合项+电阻压降），提前补偿后，PI输出更精准，电流跟踪更快；
  3. 没必要强行改变输出物理量：
    • 若想控转矩：Iq已直接对应转矩（Id=0时T∝IqT∝IqT∝Iq），无需PI输出转矩，通过速度环输出目标Iq即可；
    • 若想控转速/位置：属于外层闭环（速度环/位置环），其输出为“目标电流”（给电流环），而非直接控制转速/位置，符合“内层环快、外层环慢”的闭环设计原则（避免系统震荡）。

疑问2：FOC如何进行PID参数确定？

解答（工程整定方法，结合知乎文章与教材）：
FOC三闭环（电流环→速度环→位置环）需按“从内到外”的顺序整定，参数与电机参数强相关，步骤如下：

1. 电流环PI参数整定（最核心，响应最快）

  • 核心目标：快速跟踪电流指令，无静差、无超调；
  • 参数初值计算（基于电机参数R、L）：
    • 比例系数 Kp≈LTsK_p≈\frac{L}{T_s}Kp​≈Ts​L​（L为定子相电感，TsT_sTs​为PWM周期）；
    • 积分系数 Ki≈RLKpK_i≈\frac{R}{L}K_pKi​≈LR​Kp​（时间常数τ=LRτ=\frac{L}{R}τ=RL​，积分时间需匹配τττ）；
  • 工程试凑法：
    1. 置Ki=0K_i=0Ki​=0，逐步增大KpK_pKp​，直到电流波形出现轻微震荡；
    2. 保持KpK_pKp​不变，逐步增大KiK_iKi​，直到电流稳态无静差；
    3. 微调KpK_pKp​（减小10%-20%），消除震荡，最终电流环带宽控制在1kHz-5kHz。

2. 速度环PI参数整定

  • 核心目标：转速稳定，负载突变时无明显超调；
  • 初值参考：Kpv≈0.1KpdK_{p_v}≈0.1K_{p_d}Kpv​​≈0.1Kpd​​，Kiv≈0.01KidK_{i_v}≈0.01K_{i_d}Kiv​​≈0.01Kid​​（需结合电机转动惯量调整）；
  • 试凑法：
    1. 置Kiv=0K_{i_v}=0Kiv​​=0，增大KpvK_{p_v}Kpv​​直到转速阶跃响应出现超调（约10%-20%）；
    2. 逐步增大KivK_{i_v}Kiv​​，直到超调消失，转速稳态无静差。

3. 位置环PI参数整定

  • 核心目标：定位精准，无静差、无抖动；
  • 初值参考：Kpp≈0.01KpvK_{p_p}≈0.01K_{p_v}Kpp​​≈0.01Kpv​​，KipK_{i_p}Kip​​可设为0（位置环一般用P或PD控制，避免积分饱和）；
  • 试凑法：增大KppK_{p_p}Kpp​​直到定位无静差，若出现抖动则减小，必要时加入少量D项（KdpK_{d_p}Kdp​​）抑制震荡。

关键注意事项

  • 电机参数必须准确：若R、L测量误差大，参数初值会偏差严重，需通过实验校准电机参数；
  • 先仿真后实测：用MATLAB/Simulink搭建模型，先在仿真中整定参数，再移植到硬件，避免硬件损坏；
  • 无感FOC需额外调整：位置估算算法的误差会影响电流环，需先优化观测器，再整定PID。
